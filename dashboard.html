Data Visualization Dashboards

Overview

This project showcases full-stack analytics proficiency across two BI platforms, Power BI and Tableau, by using datasets from Kaggle to drive insights in insurance risk and human resources. The project consists of two comprehensive dashboards, each focusing on a particular business domain.

Insurance Claims Analysis

This dashboard provides an in-depth analysis of 1000 insurance policies, focusing on general claims data and a specific breakdown of fraudulent claims.

HR Analytics Dashboard

This dashboard provides a comprehensive overview of employee data, with a specific focus on attrition rate.
